# Lab Rats

A super simple text-based adventure game inspired by https://matt.might.net/articles/make-a-text-game-with-generative-ai/ to demonstrate how to use the ChatGPT API.

Run
```
pip install openai
```
and then 
```
export OPENAI_API_KEY=<your key>
```

Start the game with
```
python game.py
```
